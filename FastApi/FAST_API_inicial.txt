FAST API


Conceitos:
Verbos HTTP: Post,Get, Delete

-> Post: O método POST é usado para enviar dados ao servidor, geralmente para criar ou modificar recursos. Exemplo: fazer login, enviar um formulário, cadastrar um novo usuário ou inserir um novo registro em um banco de dados.

-> Get: O método GET é usado principalmente para solicitar dados de um servidor. Ele é usado para obter informações sem modificar o estado do servidor. Exemplo: acessar uma página, buscar uma lista de itens, carregar imagens ou qualquer recurso de leitura.
